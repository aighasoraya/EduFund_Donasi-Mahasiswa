EduFund adalah aplikasi mobile yang dirancang untuk memfasilitasi kegiatan donasi di lingkungan mahasiswa. 
Aplikasi ini menyediakan antarmuka yang sederhana dan mudah digunakan untuk mendukung penggalangan dana 
serta kegiatan sosial mahasiswa.

## Deskripsi Proyek
Proyek ini dikembangkan sebagai bagian dari tugas akademik. Fokus utama pengembangan berada pada sisi 
frontend aplikasi, mencakup perancangan antarmuka pengguna dan alur navigasi agar aplikasi mudah digunakan.

## Fitur
- Tampilan daftar program donasi
- Form donasi
- Navigasi antarlayar yang sederhana
- Desain antarmuka yang bersih dan mudah dipahami

## Teknologi yang Digunakan
- Flutter
- Dart

## Kontribusi
Saya bertanggung jawab atas pengembangan keseluruhan frontend aplikasi mobile, termasuk:
- Perancangan tampilan antarmuka
- Implementasi navigasi antarlayar
- Menjaga konsistensi dan kenyamanan penggunaan aplikasi
